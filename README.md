二次元壁纸生成器 - 运行指南
程序简介
这是一个基于 Python 的自动化壁纸工具，能够自动获取二次元图片并设置为桌面壁纸。本程序采用“便携式部署”方案，内置了独立的 Python 运行环境，无需在您的电脑系统中安装任何额外软件即可运行。
文件夹结构说明
为了确保程序稳定运行，请保持文件夹内的文件结构完整，不要随意删除或移动以下关键文件：
python.exe：核心引擎。这是内置的 Python 解释器，程序依靠它来执行代码。
python311.dll / python311.zip：Python 的基础运行库，不可或缺。
Lib / Scripts：存放 Python 第三方库（如 requests, pillow 等）的文件夹。
run.bat (或类似名称)：启动器。这是您平时用来双击运行程序的入口文件。
main.py (或类似名称)：核心脚本。存放了壁纸下载和设置的逻辑代码。
images (或类似名称)：资源文件夹。存放下载下来的壁纸图片。
requirements.txt：依赖列表。记录了程序运行所需的库名称。
运行环境要求
操作系统：Windows 7 / 10 / 11 (64位或32位，取决于您下载的 Python 版本)。
磁盘空间：建议保留至少 200MB 的可用空间（用于存放 Python 环境和缓存图片）。
网络环境：首次运行需要连接互联网，用于安装必要的 Python 依赖库。
首次运行配置（重要）
由于内置的是“嵌入式 Python”，首次运行时可能缺少包管理工具 pip。如果遇到报错，请按以下步骤操作：
检查配置文件
找到文件夹内的 python311._pth 文件，用记事本打开。确保文件最后一行是 import site，且前面没有 # 号。如果有 #，请删除它并保存。这一步是为了允许 Python 加载外部库。
安装 pip 工具
如果运行报错提示 No module named pip，请先下载 get-pip.py 脚本并放入本文件夹。
按住 Shift 键，在文件夹空白处点击鼠标右键，选择“在此处打开 Powershell 窗口”或“在此处打开命令窗口”。
输入命令 .\python.exe get-pip.py 并回车，等待安装成功提示。
安装依赖库
在同一个窗口中，输入 .\python.exe -m pip install -r requirements.txt，程序会自动下载并安装所有需要的库。
日常使用
配置完成后，您只需要双击 run.bat（或您设置的启动脚本）即可启动壁纸更新。程序会自动在后台运行，检查并更新您的桌面壁纸。
常见问题排查
报错：缺少 VCRUNTIME140.dll
这说明您的系统缺少微软常用运行库。请前往微软官网下载并安装 "Microsoft Visual C++ Redistributable for Visual Studio"。
报错：网络请求失败
请检查您的网络连接，或尝试关闭代理/VPN 后重试。
图片不更新
请检查 images 文件夹是否有写入权限，或尝试删除文件夹内的旧图片后重新运行。
维护与更新
如果您需要更新壁纸源或修改逻辑，请直接编辑 main.py 文件。如果您需要添加新的功能库，请在 Powershell 中使用 .\python.exe -m pip install 库名 进行安装。
